# Wavetable Synthesizer App
##  Learned with WolfSound video tutorials.

### The technical features of this app are:
- MVVM Architecture
- Android NDK
- Jetpack Compose
- Oboe Sound Library
- Fourier Series Understanding to Create Signals

### In this app the user can:
- Select A Waveform
- Use a Slider to Increase or Decrease Frequency of the wave
- Use a Slider to Increase or Decrease Sound
